Before use it, cd to your work directory first!
Usage:   ii -h
         ii [command] [[options] [values]]
Example: ii filetree -o tree -g f

Values:
  boolean                'true', 't', 'false' or 'f'.

Commands: 
  release                npm version patch && 
                         npm publish && 
                         git push --follow-tags
  filetree               create a file tree with "+-" and "|"
  server                 run a local static server for your 
                         current directory

Release:
  release command does not accept options.

Filetree:
  -o / --output                set name for output file. Default
                               of 'filetree'
  -t / --type                  using file type. Available in 
                               'html','markdown'/'md' and 'txt'.
  -g / -gitignore              use .gitignore file. Values with 
                               boolean. Default 't'.
  -s / --subfolder             use only the referred folder to
                               make a file tree. Value with a 
                               string of your folder inside 
                               current directory.

server:
  -p / --port                  set port for the server
  -acao / --allow-cors         set Access-Control-Allow-Origin 
                               to '*'. Values with boolean. 
                               Default 't'.
  -pd / --print-detail         print every requests of the file. 
                               Values boolean Default 't'.
  -i / --index                 set the index file. Default 
                               'index.html'.

-h / --help              show this page
